# Local Authority District to Region (April 2024) Lookup in England

A data set containing a lookup between 2024 Local Authority Districts
(LAD) and 2024 English Regions.

## Usage

``` r
lookup_ltla24_region24
```

## Format

A data frame of class "tbl" with 296 rows and 4 variables:

- ltla24_name:

  LAD name

- ltla24_code:

  LAD code

- region24_name:

  Region name

- region24_code:

  Region code
