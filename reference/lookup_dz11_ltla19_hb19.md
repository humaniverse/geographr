# Data Zone (2011) to Local Authority (2019) to Health Board (2019) Lookup

A data set containing a lookup between Data Zones, Local Authorities
(also called Council Areas) and Health Boards in Scotland

## Usage

``` r
lookup_dz11_ltla19_hb19
```

## Format

A data frame of class "tbl" with 6,976 rows and 3 variables:

- dz11_code:

  Data zone (2011) code

- ltla19_code:

  Local authortiy (2019) code

- hb19_code:

  Health board (2019) code
