# Local Authority Districts to Fire and Rescue Authorities lookup (2021).

A data set containing a lookup between April 2021 Local Authority Fire
and Rescue Authorities (FRA)

## Usage

``` r
lookup_ltla21_fra21
```

## Format

A data frame of class "tbl" with 331 rows and 4 variables:

- ltla21_name:

  LAD name

- ltla21_code:

  LAD code

- fra21_name:

  FRA name

- fra21_code:

  FRA code
