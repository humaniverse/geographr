# Lower Tier Local Authority (2023) to Upper Tier Local Authority (2023).

A data set containing a lookup between Lower Tier Local Authority (2023)
to Upper Tier Local Authority (2023).

## Usage

``` r
lookup_ltla23_utla23
```

## Format

A data frame of class "tbl" with 318 rows and 4 variables:

- ltla23_code:

  LTLA code

- ltla23_name:

  LTLA name

- utla23_code:

  UTLA code

- utla23_name:

  UTLA name
