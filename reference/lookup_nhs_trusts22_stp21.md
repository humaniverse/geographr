# England NHS Trusts (2022) to STP/ICS (2021) Lookup

A data set containing a lookup between 2022 England NHS Trusts and 2021
STP areas.

## Usage

``` r
lookup_nhs_trusts22_stp21
```

## Format

A data frame of class "tbl" with 209 rows and 4 variables:

- nhs_trust22_name:

  NHS Trust name

- nhs_trust22_code:

  NHS Trust code

- stp21_name:

  STP name

- stp21_code:

  STP code

## Source

<https://geoportal.statistics.gov.uk/>
