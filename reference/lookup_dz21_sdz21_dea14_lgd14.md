# Data Zone (2021) to Super Data Zone (2021) to District Electoral Areas (2019) to Local Government Districts (2014) Lookup

A data set containing a lookup between Data Zones (Output Areas), Super
Data Zones (LSOA), District Electoral Areas and Local Government
Districts (Local Authority) in Northern Ireland

## Usage

``` r
lookup_dz21_sdz21_dea14_lgd14
```

## Format

A data frame of class "tbl" with 3,780 rows and 8 variables:

- dz21_code:

  Data zone (2021) code (comparable to Output Areas)

- dz21_name:

  Data zone (2021) name

- sdz21_code:

  Super data zone (2021) code (comparable to LSOA / Scottish Data Zone)

- sdz21_name:

  Super data zone (2021) name

- dea14_code:

  District Electoral Areas (2019) code

- dea14_name:

  District Electoral Areas (2019) name

- lgd14_code:

  Local Government District (2014) code (comparable to Local Authority)

- lgd14_name:

  Local Government District (2014) name
