# Changes to England Local Authority codes over time (from 2019).

A data set containing a lookup of the England Local Authority changes
between 2019, 2020 and 2021.

## Usage

``` r
lookup_ltla_ltla
```

## Format

A data frame of class "tbl" with 393 rows and 18 variables:

- ltla16_name:

  2016 LAD name

- ltla16_code:

  2016 LAD code

- ltla17_name:

  2017 LAD name

- ltla17_code:

  2017 LAD code

- ltla18_name:

  2018 LAD name

- ltla18_code:

  2018 LAD code

- ltla19_name:

  2019 LAD name

- ltla19_code:

  2019 LAD code

- ltla20_name:

  2020 LAD name

- ltla20_code:

  2020 LAD code

- ltla21_name:

  2021 LAD name

- ltla21_code:

  2021 LAD code

- ltla22_name:

  2022 LAD name

- ltla22_code:

  2022 LAD code

- ltla23_name:

  2023 LAD name

- ltla23_code:

  2023 LAD code

- ltla24_name:

  2024 LAD name

- ltla24_code:

  2024 LAD code
