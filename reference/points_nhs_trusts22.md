# NHS Trusts (February 2022)

A data set containing February 2022 NHS Trusts point geometries (i.e.,
lat/long coordinates) and their associated names, codes, and open
status.

## Usage

``` r
points_nhs_trusts22
```

## Format

A data frame of class "sf" with 218 rows and 4 variables:

- nhs_trust22_name:

  NHS Trust name

- nhs_trust22_code:

  NHS Trust code

- status:

  whether the Trust is 'open' or 'closed'

- geometry:

  point coordinates

## Source

<https://digital.nhs.uk/>
