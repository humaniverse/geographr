# Rural-Urban Classifications for Local Authorities (2024) in England and Wales

A data set containing rural-urban classifications for Local Authorities
in England and Wales.

## Usage

``` r
ruc21_ltla24
```

## Format

A data frame of class "tbl" with 318 rows and 4 variables:

- ltla24_code:

  Local Authority code

- ruc21_code:

  Rural-Urban Classification code

- ruc:

  Whether the Local Authority is classified as 'Rural' or 'Urban'

- classification:

  A more detailed description of the classification
