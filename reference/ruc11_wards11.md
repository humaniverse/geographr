# Rural-Urban Classifications for wards (2011) in England and Wales

A data set containing rural-urban classifications for wards in England
and Wales.

## Usage

``` r
ruc11_wards11
```

## Format

A data frame of class "tbl" with 8,570 rows and 4 variables:

- ward11_code:

  Ward code

- ruc11_code:

  Rural-Urban Classification code

- ruc:

  Whether the ward is classified as 'Rural' or 'Urban'

- classification:

  A more detailed description of the classification
