# Sustainability and Transformation Partnerships (April 2021)

A data set containing 2021 Sustainability and Transformation
Partnerships (STP) geometries and their associated names and codes.
Boundaries are ultra generalised (500m).

## Usage

``` r
boundaries_stp21
```

## Format

A data frame of class "sf" with 42 rows and 3 variables:

- stp21_name:

  STP name

- stp21_code:

  STP code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
