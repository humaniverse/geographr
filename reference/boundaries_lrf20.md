# England and Wales Local Resilience Forums (2020)

A data set containing 2020 Local Resilience Forum geometries for England
and Wales and their associated names and codes.

## Usage

``` r
boundaries_lrf20
```

## Format

A data frame of class "sf" with 42 rows and 3 variables:

- lrf20_name:

  Local Resilience Forum name

- lrf20_code:

  Local Resilience Forum code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
