# Lower Layer Super Output Area (2011) to Middle Layer Super Output Area (2011) Lookup

A data set containing a lookup between 2011 Lower (LSOA) and Middle
Layer Super Output Areas (MSOA).

## Usage

``` r
lookup_lsoa11_msoa11
```

## Format

A data frame of class "tbl" with 41,729 rows and 4 variables:

- lsoa11_name:

  LSOA name

- lsoa11_code:

  LSOA code

- msoa11_name:

  MSOA name

- msoa11_code:

  MSOA code
