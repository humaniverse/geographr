# Local Government District (2021) to Health and Social Care Trust (2018) Lookup

A data set containing a lookup between Local Authorities (also called
Local Government Districts) and Health and Social Care Trusts in
Northern Ireland

## Usage

``` r
lookup_ltla21_hsct18
```

## Format

A data frame of class "tbl" with 11 rows and 4 variables:

- ltla21_name:

  Local Government District (2021) name

- ltla21_code:

  Local Government District (2021) code

- trust18_name:

  Health and Social Care Trust (2018) name

- trust18_code:

  Health and Social Care Trust (2018) code
