# Local Authority Districts to VCSEP regions lookup (2021).

A data set containing a lookup between December 2021 Local Authority
Districts and Voluntary and Community Sector Emergency Partnership
regions

## Usage

``` r
lookup_ltla21_vcsep22
```

## Format

A data frame of class "tbl" with 309 rows and 3 variables:

- ltla21_name:

  LAD name

- ltla21_code:

  LAD code

- vcsep_region:

  VCSEP region name
