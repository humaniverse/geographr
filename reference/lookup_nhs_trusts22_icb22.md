# England NHS Trusts (2022) to ICB (2022) Lookup

A data set containing a lookup between 2022 England NHS Trusts and 2022
Integrate Care Boards (ICBs).

## Usage

``` r
lookup_nhs_trusts22_icb22
```

## Format

A data frame of class "tbl" with 1,321 rows and 3 variables:

- icb22_code:

  ICB code

- nhs_trust22_code:

  NHS Trust code

- proportion_trust_came_from_icb:

  The proportion of patients from all admissions from a given Trust that
  came from a given ICB
