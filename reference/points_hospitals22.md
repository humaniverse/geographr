# NHS England Hopsitals (July 2022)

A data set containing July 2022 NHS England hospital point geometries
(i.e., lat/long coordinates) and their associated names.

## Usage

``` r
points_hospitals22
```

## Format

A data frame of class "sf" with 1,061 rows and 2 variables:

- hospital22_name:

  NHS England hospital

- geometry:

  point coordinates

## Source

<https://digital.nhs.uk/>
