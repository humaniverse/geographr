# Rural-Urban Classifications for Lower Layer Super Output Areas (2011) in England and Wales

A data set containing rural-urban classifications for LSOAs in England
and Wales.

## Usage

``` r
ruc11_lsoa11
```

## Format

A data frame of class "tbl" with 34,753 rows and 4 variables:

- lsoa11_code:

  LSOA code

- ruc11_code:

  Rural-Urban Classification code

- ruc:

  Whether the LSOA is classified as 'Rural' or 'Urban'

- classification:

  A more detailed description of the classification
