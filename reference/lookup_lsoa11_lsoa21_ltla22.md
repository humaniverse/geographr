# Lower Layer Super Output Area (2011) to Lower Layer Super Output Area (2021) to LAD (December 2022) Lookup

A data set containing a lookup between 2011 Lower Super Output Areas
(LSOA) and 2021 Lower Super Output Areas (LSOA) and 2022 Local Authority
Districts (LAD).

## Usage

``` r
lookup_lsoa11_lsoa21_ltla22
```

## Format

A data frame of class "tbl" with 35,795 rows and 8 variables:

- lsoa11_name:

  LSOA 11 name

- lsoa11_code:

  LSOA 11 code

- change_code:

  Change code containing U (unchanged from 2011 to 2021), M (merged), S
  (split) and X (irregular and fragmented change)

- lsoa21_name:

  LSOA 21 name

- lsoa21_code:

  LSOA 21 code

- ltla22_name:

  LTLA name

- ltla22_code:

  LTLA code
