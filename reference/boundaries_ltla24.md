# Local Authority Districts (July 2024)

A data set containing 2024 Local Authority Districts (LAD) geometries
and their associated names and codes. Boundaries are Ultra Generalised
(500m) - clipped to the coastline (Mean High Water mark).

## Usage

``` r
boundaries_ltla24
```

## Format

A data frame of class "sf" with 361 rows and 3 variables:

- ltla24_name:

  LAD name

- ltla24_code:

  LAD code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
