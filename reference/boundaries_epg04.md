# Emergency Preparedness Groups, Northern Ireland (2004)

A data set containing 2004 NI Emergency Preparedness Group geometries
and their associated names. Boundaries created by dissolving LGD
geometries, from ONS.

## Usage

``` r
boundaries_epg04
```

## Format

A data frame of class "sf" with 3 rows and 2 variables:

- epg04_name:

  Emergency Preparedness Greoup name

- geometry:

  multipolygon coordinates
