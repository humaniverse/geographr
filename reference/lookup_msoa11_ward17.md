# MSOA (2011) to Ward (2017) Lookup

A data set containing a lookup between 2011 MSOAs and 2021 wards in
England and Wales.

## Usage

``` r
lookup_msoa11_ward17
```

## Format

A data frame of class "tbl" with 13,873 rows and 2 variables:

- msoa11_code:

  MSOA 2011 code

- ward17_code:

  Ward 2017 code

<https://geoportal.statistics.gov.uk/>
