# MSOA (2021) to Ward (2017) Lookup

A data set containing a lookup between 2021 MSOAs and 2021 wards in
England and Wales.

## Usage

``` r
lookup_msoa21_ward17
```

## Format

A data frame of class "tbl" with 13,865 rows and 2 variables:

- msoa21_code:

  MSOA 2021 code

- ward17_code:

  Ward 2017 code

## Source

<https://geoportal.statistics.gov.uk/>
