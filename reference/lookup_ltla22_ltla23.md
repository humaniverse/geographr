# Local Authority Districts (2022) to Local Authority Districts (2023).

A data set containing a lookup between 2022 Local Authority Districts
and 2023 Local Authority Districts.

## Usage

``` r
lookup_ltla22_ltla23
```

## Format

A data frame of class "tbl" with 374 rows and 4 variables:

- ltla22_code:

  LAD 22 code

- ltla22_name:

  LAD 22 name

- ltla23_code:

  LAD 23 code

- ltla23_name:

  LAD 23 name
