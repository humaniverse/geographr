# Clinical Commissioning Groups (April 2021)

A data set containing 2021 Clinical Commissioning Group (CCG) geometries
and their associated names and codes. Boundaries are ultra generalised
(500m).

## Usage

``` r
boundaries_ccg21
```

## Format

A data frame of class "sf" with 106 rows and 3 variables:

- ccg21_name:

  CCG name

- ccg21_code:

  CCG code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
