# Integrated Care Boards (2022)

A data set containing July 2022 NHS Integrated Care Board geometries and
their associated names and codes.

## Usage

``` r
boundaries_icb22
```

## Format

A data frame of class "sf" with 42 rows and 3 variables:

- icb22_name:

  Integrated Care Board name

- icb22_code:

  Integrated Care Board code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
