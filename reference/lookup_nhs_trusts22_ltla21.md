# England NHS Trusts (2022) to LTLA (2021) Lookup

A data set containing a lookup between 2022 England NHS Trusts and 2021
LTLA areas.

## Usage

``` r
lookup_nhs_trusts22_ltla21
```

## Format

A data frame of class "tbl" with 574,104 rows and 4 variables:

- ltla21_code:

  LTLA code

- nhs_trust22_code:

  NHS Trust code

- proportion_trust_came_from_ltla:

  The proportion of patients from all admissions from a given Trust that
  came from a given LTLA
