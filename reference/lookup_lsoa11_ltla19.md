# Lower Layer Super Output Area (2011) to LAD (December 2019) Lookup

A data set containing a lookup between 2011 Lower Super Output Areas
(LSOA) and 2019 Local Authority Districts (LAD).

## Usage

``` r
lookup_lsoa11_ltla19
```

## Format

A data frame of class "tbl" with 34,753 rows and 4 variables:

- lsoa11_name:

  LSOA name

- lsoa11_code:

  LSOA code

- ltla19_name:

  LAD name

- ltla19_code:

  LAD code
