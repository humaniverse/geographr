# Lower Layer Super Output Area (2021) to Ward (2024) to Local Authority District (2024) Lookup

A data set containing a lookup between 2021 Lower Super Output Areas
(LSOA) and 2024 Wards and Local Authority Districts.

## Usage

``` r
lookup_lsoa21_ward24_ltla24
```

## Format

A data frame of class "tbl" with 35,672 rows and 6 variables:

- lsoa21_name:

  LSOA name

- lsoa21_code:

  LSOA code

- ward24_name:

  Ward name

- ward24_code:

  Ward code

- ltla24_name:

  LAD name

- ltla24_code:

  LAD code
