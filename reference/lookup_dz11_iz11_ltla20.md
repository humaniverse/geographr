# Data Zone (2011) to Intermediate Zone (2011) to LAD (December 2019) Lookup

A data set containing a lookup between Data Zones, Intermediate Zones
and Council Areas in Scotland

## Usage

``` r
lookup_dz11_iz11_ltla20
```

## Format

A data frame of class "tbl" with 6,976 rows and 6 variables:

- dz11_name:

  DZ name

- dz11_code:

  DZ code

- iz11_name:

  IZ name

- iz11_code:

  IZ code

- ltla20_name:

  LAD name

- ltla20_code:

  LAD code
