# Super Output Areas (2011)

A data set containing 2011 Northern Ireland Super Output Area geometries
and their associated names and codes as used following the 2011 Census
results.

## Usage

``` r
boundaries_soa11
```

## Format

A data frame of class "sf" with 890 rows and 3 variables:

- soa11_name:

  Super Output Area name

- soa11_code:

  Super Output Area code

- geometry:

  multipolygon coordinates
