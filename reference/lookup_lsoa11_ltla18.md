# Lower Layer Super Output Area (2011) to LAD (December 2018) Lookup

A data set containing a lookup between 2011 Lower Super Output Areas
(LSOA) and 2018 Local Authority Districts (LAD).

## Usage

``` r
lookup_lsoa11_ltla18
```

## Format

A data frame of class "tbl" with 32,844 rows and 4 variables:

- lsoa11_name:

  LSOA name

- lsoa11_code:

  LSOA code

- ltla18_name:

  LAD name

- ltla18_code:

  LAD code
