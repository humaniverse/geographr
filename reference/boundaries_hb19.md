# Health Boards (2019)

A data set containing 2019 NHS Health Board geometries and their
associated names and codes.

## Usage

``` r
boundaries_hb19
```

## Format

A data frame of class "sf" with 14 rows and 3 variables:

- hb19_name:

  Health Board name

- hb19_code:

  Health Board code

- geometry:

  multipolygon coordinates

## Source

<https://spatialdata.gov.scot/>
