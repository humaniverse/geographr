# LSOA (2021) to SICBL to ICB to Cancer Alliances to LAD (April 2024) Lookup

A data set containing a lookup between LSOA (2021), Sub Integrated Care
Board Locations (SICBL) and Integrated Care Boards (ICB).

## Usage

``` r
lookup_lsoa21_icb24
```

## Format

A data frame of class "tbl" with 33,755 rows and 6 variables:

- lsoa21_name:

  LSOA name

- lsoa21_code:

  LSOA code

- sicbl24_name:

  Sub Integrated Care Board Location name

- sicbl24_code:

  Sub Integrated Care Board Location code

- icb24_name:

  Integrated Care Board name

- icb24_code:

  Integrated Care Board code
