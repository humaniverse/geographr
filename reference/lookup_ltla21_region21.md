# Local Authority District to Region (April 2021) Lookup in England

A data set containing a lookup between 2021 Local Authority Districts
(LAD) and 2021 English Regions.

## Usage

``` r
lookup_ltla21_region21
```

## Format

A data frame of class "tbl" with 317 rows and 4 variables:

- ltla21_name:

  LAD name

- ltla21_code:

  LAD code

- region21_name:

  Region name

- region21_code:

  Region code
