# Intermediate Zones (2011)

A data set containing 2011 Scottish Intermediate Zone geometries and
their associated names and codes.

## Usage

``` r
boundaries_iz11
```

## Format

A data frame of class "sf" with 1,279 rows and 3 variables:

- iz11_name:

  Intermediate Zone name

- iz11_code:

  Intermediate Zone code

- geometry:

  multipolygon coordinates

## Source

<https://spatialdata.gov.scot/>
