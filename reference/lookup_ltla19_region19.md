# Local Authority District to Region (December 2019) Lookup in England

A data set containing a lookup between 2019 Local Authority Districts
(LAD) and 2019 English Regions.

## Usage

``` r
lookup_ltla19_region19
```

## Format

A data frame of class "tbl" with 317 rows and 4 variables:

- ltla19_name:

  LAD name

- ltla19_code:

  LAD code

- region19_name:

  Region name

- region19_code:

  Region code
