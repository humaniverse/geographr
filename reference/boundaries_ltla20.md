# Local Authority Districts (December 2020)

A data set containing 2020 Local Authority Districts (LAD) geometries
and their associated names and codes. Boundaries are Ultra Generalised
(500m) - clipped to the coastline (Mean High Water mark).

## Usage

``` r
boundaries_ltla20
```

## Format

A data frame of class "sf" with 379 rows and 3 variables:

- ltla20_name:

  LAD name

- ltla20_code:

  LAD code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
