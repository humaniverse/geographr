# Ward (December 2021) to LTLA (December 2021) Lookup

A data set containing a lookup between 2021 wards and 2021 Lower Tier
Local Authority Districts (LAD).

## Usage

``` r
lookup_ward21_ltla21
```

## Format

A data frame of class "tbl" with 8,694 rows and 4 variables:

- ward21_name:

  MSOA name

- ward21_code:

  MSOA code

- ltla21_name:

  LAD name

- ltla21_code:

  LAD code
