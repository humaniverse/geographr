# Super Data Zones (2021)

A data set containing 2021 Northern Ireland Super Data Zones geometries
and their associated names and codes as used following the 2021 Census
results.

## Usage

``` r
boundaries_sdz21
```

## Format

A data frame of class "sf" with 850 rows and 3 variables:

- sdz21_name:

  Super Data Zone name

- sdz21_code:

  Super Data Zone code

- geometry:

  multipolygon coordinates
