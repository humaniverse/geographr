# LSOA (2011) to Ward (2017) Lookup

A data set containing a lookup between 2011 LSOAs and 2021 wards in
England and Wales.

## Usage

``` r
lookup_lsoa11_ward17
```

## Format

A data frame of class "tbl" with 34,753 rows and 2 variables:

- lsoa11_code:

  LSOA 2011 code

- ward17_code:

  Ward 2017 code

## Source

<https://geoportal.statistics.gov.uk/>
