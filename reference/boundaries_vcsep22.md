# VCSEP regions (2022).

A data set containing geometries for the Voluntary and Community Sector
Emergency Partnership regions

## Usage

``` r
boundaries_vcsep22
```

## Format

A data frame of class "tbl" with 5 rows and 2 variables:

- vcsep_region:

  VCSEP region name

- geometry:

  multipolygon coordinates
