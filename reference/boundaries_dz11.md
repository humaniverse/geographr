# Data Zones (2011)

A data set containing 2011 Scottish Data Zone geometries and their
associated names and codes.

## Usage

``` r
boundaries_dz11
```

## Format

A data frame of class "sf" with 6,976 rows and 3 variables:

- dz11_name:

  Data Zone name

- dz11_code:

  Data Zone code

- geometry:

  multipolygon coordinates

## Source

<https://spatialdata.gov.scot/>
