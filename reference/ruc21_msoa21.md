# Rural-Urban Classifications for Middle Layer Super Output Areas (2021) in England and Wales

A data set containing rural-urban classifications for MSOAs in England
and Wales.

## Usage

``` r
ruc21_msoa21
```

## Format

A data frame of class "tbl" with 7,264 rows and 4 variables:

- msoa21_code:

  MSOA code

- ruc21_code:

  Rural-Urban Classification code

- ruc:

  Whether the MSOA is classified as 'Rural' or 'Urban'

- classification:

  A more detailed description of the classification

## Source
