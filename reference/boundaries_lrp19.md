# Scotland Local Resilience Partnerships (2019)

A data set containing 2019 Local Resilience Partnership geometries for
Scotland and their associated names and codes.

## Usage

``` r
boundaries_lrp19
```

## Format

A data frame of class "sf" with 12 rows and 3 variables:

- lrp19_name:

  Local Resilience Partnership name

- lrp19_code:

  Local Resilience Partnership code

- geometry:

  multipolygon coordinates

## Source

<https://data.gov.uk/>
