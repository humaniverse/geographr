# Counties and Unitary Authorities (December 2020)

A data set containing 2020 County and Unitary Authority (UA) geometries
and their associated names and codes. Boundaries are Ultra Generalised
(500m) - clipped to the coastline (Mean High Water mark).

## Usage

``` r
boundaries_utla20
```

## Format

A data frame of class "sf" with 216 rows and 3 variables:

- utla20_name:

  county and UA name

- utla20_code:

  county and UA code

- geometry:

  multipolygon coordinates
