# Regions (December 2021)

A data set containing December 2021 England Region geometries and their
associated names and codes.

## Usage

``` r
boundaries_region21
```

## Format

A data frame of class "sf" with 890 rows and 3 variables:

- region21_name:

  Region name

- region21_code:

  Region code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
