# Local Authority Districts (December 2019)

A data set containing 2019 Local Authority Districts (LAD) geometries
and their associated names and codes. Boundaries are Ultra Generalised
(500m) - clipped to the coastline (Mean High Water mark).

## Usage

``` r
boundaries_ltla19
```

## Format

A data frame of class "sf" with 382 rows and 3 variables:

- ltla19_name:

  LAD name

- ltla19_code:

  LAD code

- geometry:

  multipolygon coordinates

## Source

<https://geoportal.statistics.gov.uk/>
