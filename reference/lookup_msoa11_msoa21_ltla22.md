# Middle Layer Super Output Area (2011) to Middle Layer Super Output Area (2021) to LAD (December 2022) Lookup

A data set containing a lookup between 2011 Middle Super Output Areas
(MSOA) and 2021 Middle Super Output Areas (MSOA) and 2022 Local
Authority Districts (LAD).

## Usage

``` r
lookup_msoa11_msoa21_ltla22
```

## Format

A data frame of class "tbl" with 7.201 rows and 6 variables:

- msoa11_name:

  MSOA 11 name

- msoa11_code:

  MSOA 11 code

- msoa21_name:

  MSOA 21 name

- msoa21_code:

  MSOA 21 code

- ltla22_name:

  LAD name

- ltla22_code:

  LAD code
