# Rural-Urban Classifications for Middle Layer Super Output Areas (2011) in England and Wales

A data set containing rural-urban classifications for MSOAs in England
and Wales.

## Usage

``` r
ruc11_msoa11
```

## Format

A data frame of class "tbl" with 7,201 rows and 4 variables:

- msoa11_code:

  MSOA code

- ruc11_code:

  Rural-Urban Classification code

- ruc:

  Whether the MSOA is classified as 'Rural' or 'Urban'

- classification:

  A more detailed description of the classification
