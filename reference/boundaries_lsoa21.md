# Lower Layer Super Output Areas (2021)

A data set containing 2021 Lower Super Output Area (LSOA) geometries and
their associated names and codes. Full clipped boundaries were
simplified using the Visvalingam algorithm set to a weighting of 0.7
with a retaining proportion of 0.895 points.

## Usage

``` r
boundaries_lsoa21
```

## Format

A data frame of class "sf" with 35,672 rows and 3 variables:

- lsoa21_name:

  LSOA name

- lsoa21_code:

  LSOA code

- geometry:

  multipolygon coordinates

## Source
