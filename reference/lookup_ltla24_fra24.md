# Local Authority Districts to Fire and Rescue Authorities lookup (2024).

A data set containing a lookup between April 2024 Local Authority Fire
and Rescue Authorities (FRA)

## Usage

``` r
lookup_ltla24_fra24
```

## Format

A data frame of class "tbl" with 318 rows and 4 variables:

- ltla24_name:

  LAD name

- ltla24_code:

  LAD code

- fra24_name:

  FRA name

- fra24_code:

  FRA code
