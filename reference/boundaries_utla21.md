# Counties and Unitary Authorities (May 2021)

A data set containing 2021 County and Unitary Authority (UA) geometries
and their associated names and codes. Boundaries are Ultra Generalised
(500m) - clipped to the coastline (Mean High Water mark).

## Usage

``` r
boundaries_utla21
```

## Format

A data frame of class "sf" with 217 rows and 3 variables:

- utla21_name:

  county and UA name

- utla21_code:

  county and UA code

- geometry:

  multipolygon coordinates
